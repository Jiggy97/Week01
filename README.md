1. 소요 기술, 기술적 난이도, 구현가능성 등
2. 창의성(개선도 가능), 기술성, 실용성(사업성, 수익성)

'''
* 알람 앱
   - 설정한 시간에 알림이 울리고 끄지 않거나 3번 이상 연장하면
      미리 지정해 놓은 번호로 연락을 보내주는 기능
* 적절한 모임 장소 추천 앱
   - 여러 사람이 모일 때 각자 위치를 기반으로 최선의 모임 장소를 추천해주는 기능
'''

**->창의성은 좋으나 실용적이지 못함**

'''
* 도서대여 서비스 앱
   - 우리집은 도서관(실제 있는 어플) : 유료 대여, 5권 이상부터 빌릴 수 있음, 
                                                    결제 완료 후 평균 3일 안에 집 앞 배달
* 주차 자리 찾아주는 앱
   - 주차가 가능한 자리, 주차 요금, 목적지와 거리 등
      정보를 제공하고 주차가 가능한 자리까지 안내해주는 기능
'''

**-> 실용성이 굉장히 좋으나 학생 신분으로 구현 가능성이 낮음.**

'''
* 달리기 도우미 앱
   - 본인이 있는 위치에서 몇키로를 달릴지 목표를 설정하면
      해당 키로수를 충족 시켜주는 달리기 코스를 자동으로 짜주고,
      완주를 하기 전까지 코스의 마킹이 지워지지 않는 기능
* 자동 답신 앱
   - 바빠서 핸드폰을 만지지 못하거나, 핸드폰을 잠시 다른 곳에 놓아야 하는 상황에서
      앱을 이용해 미리 선정한 내용의 텍스트를 자동으로 답신하는 기능
'''

**-> 실용성, 구현 가능성 등 좋지만 이미 관련 분야 앱들이 완벽한 완성도를 소유**

'''
* 여행 앱
    - 평소 코로나로 인해 가지 못하는 여행지의 위치에 있는 앱 사용자와 컨택하여
       실시간으로 사진을 공유받고 채팅기능을 통해 소통하는 앱
'''

     어플을 구상할 수 있고, 기존에 있는 SNS앱들과도 차별을 둘 수 있을거같아 채택**
